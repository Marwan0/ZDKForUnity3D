OpenNISplitScreenSkeletonController Class

Updates 2 OpenNISkeleton objects with the first 2 users to calibrate.

public class OpenNISplitScreenSkeletonController : UnityEngine.MonoBehaviour

⊟ Remarks

After 2 users calibrate successfully all other users will be ignored. In case one of the calibrated users leaves the scene a new user will be able to calibrate. Requires an OpenNIUserTracker
