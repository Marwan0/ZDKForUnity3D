OpenNISkeleton Class

The OpenNISkeleton represents skeleton data for a single user. When an OpenNISkeleton is referenced by a controller (see OpenNISingleSkeletonController) the transforms it contains will reflect the OpenNI skeleton associated with it. The mapping from OpenNI skeleton data to the Unity Transforms is different for every OpenNISkeleton instance and depends on its exact configuration

public class OpenNISkeleton : UnityEngine.MonoBehaviour

⊟ Remarks

Joints that aren't provided by OpenNI will not be modified.
